# Anime Style LoRA for SDXL

High-quality anime style LoRA for SDXL base model. Produces clean, professional anime-style images.

## Features
- Clean line art
- Vibrant colors
- Professional anime aesthetics
- Works with various prompts

## Usage
Recommended weight: 0.7-0.9. Works best with detailed prompts describing character features and scene composition.

### Download
```bash
wget -P "$UI_HOME/models/loras" \
  https://huggingface.co/community-models/anime-lora-sdxl/resolve/main/anime_v2.safetensors
```
