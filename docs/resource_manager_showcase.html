Quick Start

  1. Start the application: 
    docker-compose up -d
  2. Open browser to http://localhost:5000
  3. Click the "üì¶ Resource Manager" tab
  4. Browse resources using filters or search
  5. Select resources by clicking "Select" button
  6. Enter SSH connection (e.g., root@host:port)
  7. Click "Install Selected" to deploy
